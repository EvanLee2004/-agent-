# Accounting Skill

## 概述

记账技能是智能会计的核心能力，负责：
- 解析用户输入的记账信息
- 执行数据库写入
- 异常检测

## 与智能会计的关系

智能会计（accountant_agent.py）是单 Agent 系统，记账技能是其内置能力之一。

## 功能

### 记账信息解析

从自然语言中提取：
- 日期（YYYY-MM-DD）
- 金额（数字）
- 类型（收入/支出）
- 说明

### 异常检测

- 金额 > 50000：标注"需审核"
- 金额 < 10：标注"金额过小"

### 经验记录

每次成功记账后自动记录经验到记忆系统。

## 使用方式

在智能会计中，用户直接说"报销500元"即可触发记账。
