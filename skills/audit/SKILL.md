# Audit Skill

## 概述

审计技能是智能会计的核心能力，负责：
- 检查记账合规性
- 标注异常情况
- 提供审核反馈

## 与智能会计的关系

智能会计（accountant_agent.py）是单 Agent 系统，审计技能是其内置能力之一。
当前 v1.0 版本中，审计功能集成在记账流程中自动执行。

## 功能

### 合规性检查

- 金额是否在合理范围
- 必填字段是否完整
- 描述是否清晰

### 异常标注

- 金额 > 50000：需审核
- 金额 < 10：金额过小

### 审核反馈

发现问题后提供明确的修改建议。

## 使用方式

在智能会计中，记账时会自动进行审计检查。
